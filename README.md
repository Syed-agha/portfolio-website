Syed Agha | Portfolio
A modern, fully animated single-page portfolio website built with HTML, CSS, and JavaScript — featuring a light forest aesthetic inspired by nature.

🔗 Live Site
[Visit Portfolio Website](https://syed-agha.github.io/portfolio-website)

📂 File Structure
portfolio-website/
├── index.html       → All sections in one page (Home, About, Projects, Contact)
├── style.css        → All styles and animations
├── script.js        → All JavaScript functionality
├── photo.PNG        → Profile photo
└── README.md

🛠️ Technologies Used

1.HTML5
2.CSS3
3.JavaScript (Vanilla)
4.Formspree — Contact form backend
5.Google Fonts — Cormorant & Outfit

✨ Features

1.🌿 Light forest aesthetic with nature-inspired color palette
2.🍃 Falling leaves canvas animation
3.🖱️ Custom cursor with smooth trailing effect
4.📸 3D floating photo with orbiting ring and mouse parallax
5.🏷️ Floating tags on hero photo (CS Student, Full-Stack Dev, Problem Solver)
6.📊 Animated skill bars triggered on scroll
7.🃏 3D tilt effect on project cards
8.📜 Scroll reveal animations throughout
9.💧 Animated blob backgrounds
10.📬 Fully working contact form (sends emails via Formspree)
11.📱 Fully responsive design for mobile and desktop
12.⬆️ Scroll to top button

📋 Sections

Home — Hero with 3D photo, badge, stats bar
About — Bio, education timeline, skill bars
Projects — Portfolio Website, Event Management System, File Management System
Contact — Contact info + working contact form
