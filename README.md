# Syed Agha | Portfolio

A modern, fully animated single-page portfolio website built with HTML, CSS, and JavaScript — featuring a light forest aesthetic inspired by nature.

## 🔗 Live Site

[Visit Portfolio Website](https://syed-agha.github.io/portfolio-website)

## 📂 File Structure

```
portfolio-website/
├── index.html       → All sections in one page (Home, About, Projects, Contact)
├── style.css        → All styles and animations
├── script.js        → All JavaScript functionality
├── photo.PNG        → Profile photo
└── README.md
```

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Formspree](https://formspree.io) — Contact form backend
- Google Fonts — Cormorant & Outfit

## ✨ Features

- 🌿 Light forest aesthetic with nature-inspired color palette
- 🍃 Falling leaves canvas animation
- 🖱️ Custom cursor with smooth trailing effect
- 📸 3D floating photo with orbiting ring and mouse parallax
- 🏷️ Floating tags on hero photo (CS Student, Full-Stack Dev, Problem Solver)
- 📊 Animated skill bars triggered on scroll
- 🃏 3D tilt effect on project cards
- 📜 Scroll reveal animations throughout
- 💧 Animated blob backgrounds
- 📬 Fully working contact form (sends emails via Formspree)
- 📱 Fully responsive design for mobile and desktop
- ⬆️ Scroll to top button

## 📋 Sections

- **Home** — Hero with 3D photo, badge, stats bar
- **About** — Bio, education timeline, skill bars
- **Projects** — Portfolio Website, Event Management System, File Management System
- **Contact** — Contact info + working contact form
